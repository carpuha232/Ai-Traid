# 🎨 GUI Компоненты Scalping Bot

## Доступные интерфейсы

### 1. ⭐ Modern Window (НОВЫЙ - РЕКОМЕНДУЕТСЯ)
**Файл:** `modern_window.py`

Самый современный и красивый интерфейс:
- ✅ Профессиональный дизайн в стиле Binance Pro
- ✅ Темная премиум палитра
- ✅ Минималистичный flat design
- ✅ Градиенты и анимации
- ✅ Современные шрифты (Segoe UI)
- ✅ Интуитивная структура
- ✅ Красивые кнопки без объемных эффектов

**Использование:**
```python
from gui.modern_window import ModernScalpingGUI
# Бот автоматически использует этот интерфейс
```

### 2. 🎯 Optimal Window
**Файл:** `optimal_window.py`

Простой и функциональный интерфейс:
- Компактная структура
- Все данные на экране
- Классический темный стиль
- Удобная организация

### 3. 💎 Premium Window
**Файл:** `premium_window.py`

Продвинутый интерфейс с таблицами:
- Таблицы как на бирже
- История торговли
- Детальная информация
- Профессиональные панели

## Быстрый старт

### Запуск бота
```bash
python main.py
```
Бот автоматически откроет **Modern Window** интерфейс.

### Просмотр демо дизайна
```bash
python gui/preview_design.py
```

### Смена интерфейса

Чтобы изменить интерфейс, отредактируйте `main.py`:

```python
# Для Modern Window (текущий)
from gui.modern_window import ModernScalpingGUI

# Для Optimal Window
from gui.optimal_window import OptimalScalpingGUI

# Для Premium Window  
from gui.premium_window import PremiumScalpingGUI

# Затем в классе:
self.gui = ModernScalpingGUI(self.root, self.config)
```

## Новый дизайн - что изменилось?

### Цветовая схема
- **Фон:** глубокий синий (#0a0e27)
- **Панели:** темно-синий (#141b2d)
- **Profit:** ярко-зеленый (#00ff88)
- **Loss:** ярко-красный (#ff4757)
- **Акцент:** синий (#4facfe)

