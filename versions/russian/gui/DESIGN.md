# 🎨 Дизайн Scalping Bot GUI

## Новый Modern Design

### Цветовая палитра

**Темная профессиональная тема:**
- Основной фон: `#0a0e27` - глубокий синий
- Панели: `#141b2d` - темно-синий
- Заголовки: `#0f1419` - почти черный
- Границы: `#1f2e3e` - сине-серый

**Акценты:**
- Profit: `#00ff88` - ярко-зеленый
- Loss: `#ff4757` - ярко-красный
- Info: `#4facfe` - синий акцент
- Warning: `#ffa726` - оранжевый
- Premium: `#b794f6` - фиолетовый

### Особенности дизайна

1. **Минималистичный стиль** - как у профессиональных платформ (Binance Pro, Bybit)
2. **Градиенты** - плавные переходы цветов
3. **Современные шрифты** - Segoe UI для текста, Consolas для моноширинного
4. **Плоские кнопки** - без объемных эффектов
5. **Чистые линии** - без рельефов и бордюров (flat design)

### Цветовая логика

**Зеленый** - прибыль, LONG, успех
**Красный** - убыток, SHORT, ошибки  
**Синий** - информация, акцент
**Оранжевый** - предупреждения
**Фиолетовый** - премиум функции

### Интерактивность

- Кнопки имеют эффект при наведении
- Таблицы поддерживают выбор
- Автоматическая прокрутка лога
- Live обновление всех данных

### Шрифты

- **Segoe UI** - основной текст (readable, modern)
- **Consolas** - моноширинный (для чисел и логов)

### Иконки

Используются emoji для визуального разделения:
- 🚀 - запуск
- 💰 - деньги/счет
- 📊 - статистика
- 📈 - графики/сигналы
- 📜 - история
- 📋 - лог
- 🟢 - LONG
- 🔴 - SHORT
- ✅ - успех
- ❌ - ошибка
